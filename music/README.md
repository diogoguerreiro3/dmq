# Music Files

Add movie soundtracks here organized by movie folders.

## Folder Structure

```
music/
├── Frozen/
│   ├── Frozen - 01 - Let It Go.mp3
│   ├── Frozen - 02 - Do You Want to Build a Snowman.mp3
│   └── ...
├── Toy Story/
│   ├── Toy Story - 01 - You've Got a Friend in Me.mp3
│   └── ...
└── ...
```

## Requirements

- **Format**: MP3
- **Bitrate**: 192 kbps or higher recommended
- **Folder name**: Must match the movie name in `musics.json`

## Important Notes

- Normalize audio levels to avoid volume inconsistencies
- Remove long silences at the beginning/end of files

## Copyright

Please respect copyright laws. Only use music you have the rights to use.
