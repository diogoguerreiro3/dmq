# Movie Posters

Add movie poster images here in `.jpg` format.

## Naming Convention

The filename must match exactly the movie name in `musics.json`:

```
Frozen.jpg
Moana.jpg
Toy Story.jpg
The Lion King.jpg
```

## Requirements

- **Format**: JPG
- **Recommended size**: 500x750 pixels or similar aspect ratio
- **Quality**: High quality images preferred

## Where to Find Posters

- The Movie Database (TMDB)
- IMDb
- TVDB
